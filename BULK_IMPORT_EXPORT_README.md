# 批量商品导入/导出功能使用说明

## 功能概述

为商城后台管理系统添加了完整的批量商品导入/导出功能，支持Excel格式的商品数据批量管理，提升商品管理效率。

- ✅ 分步骤导入流程
- ✅ 导入结果报告

### 批量导出功能
- ✅ 灵活的筛选条件配置

- ✅ 多种导出格式支持(Excel/CSV)
- ✅ 实时预览导出数据量
- ✅ 进度监控和取消机制

### 性能优化
- ✅ 虚拟滚动支持大数据量展示
- ✅ 文件处理性能监控
- ✅ 智能错误处理和重试机制
- ✅ 内存使用优化

## 使用方法

### 批量导入
1. 在商品列表页面点击"批量导入"按钮
2. 按照提示下载Excel模板
3. 填写商品数据并上传文件
4. 预览数据并确认导入

### 批量导出
1. 在商品列表页面点击"批量导出"按钮
2. 设置筛选条件和导出字段
3. 确认导出配置并开始导出
4. 下载生成的文件

## 支持的数据字段

### 必填字段
- 商品名称 (name)
- 商品货号 (productSn) - 必须唯一
- 商品分类 (categoryName)
- 品牌名称 (brandName)
- 商品价格 (price)
- 库存数量 (stock)

### 可选字段
- 商品副标题 (subTitle)
- 市场价格 (originalPrice)
- 计量单位 (unit)
- 商品重量 (weight)
- 商品描述 (description)
- 关键词 (keywords)
- 排序值 (sort)
- 上架状态 (publishStatus)

## 数据验证规则

1. **文件格式**：支持.xlsx和.xls格式
2. **文件大小**：最大10MB
3. **数据量限制**：单次最多导入1000条
4. **字段验证**：
   - 商品名称：2-140字符
   - 商品货号：必须唯一
   - 价格：必须大于0
   - 库存：必须大于等于0
   - 分类和品牌：必须在系统中存在

## 后端API要求

需要后端实现以下API接口：

1. `GET /api/product/import/template` - 下载导入模板
2. `POST /api/product/import/validate` - 验证上传文件
3. `POST /api/product/import/confirm` - 确认批量导入
4. `GET /api/product/export/options` - 获取导出配置选项
5. `POST /api/product/export` - 执行批量导出

## 注意事项

1. **浏览器兼容性**：需要支持现代浏览器(Chrome 60+, Firefox 55+, Safari 12+)
2. **内存使用**：处理大文件时会占用较多内存，建议单文件不超过10MB
3. **网络环境**：大文件上传需要稳定的网络环境
4. **数据备份**：建议在大批量操作前备份重要数据
