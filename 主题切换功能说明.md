# 主题切换功能说明

## 功能概述
已成功为mall-admin-web项目添加了完整的明暗主题切换功能，支持一键在浅色和深色主题之间切换。

## 功能特性

### 1. 主题状态管理
- 使用Vuex管理主题状态
- 主题状态持久化到Cookie中
- 支持`toggleTheme`和`setTheme`两种操作方式

### 2. 主题样式系统
- 定义了完整的CSS变量系统
- 支持浅色主题（light）和深色主题（dark）
- 涵盖背景色、文字色、边框色、侧边栏、导航栏、卡片、表格等所有UI元素

### 3. 主题切换组件
- 位于导航栏右侧的主题切换按钮
- 使用月亮/太阳图标指示当前主题
- 鼠标悬停显示提示文字
- 点击即可切换主题

### 4. 响应式主题应用
- 根组件动态应用主题数据属性
- 所有子组件自动继承主题样式
- 平滑的过渡动画效果

## 使用方法

### 1. 切换主题
点击导航栏右侧的主题切换按钮（月亮/太阳图标）即可在明暗主题间切换。

### 2. 编程方式切换
```javascript
// 切换主题
this.$store.dispatch('toggleTheme')

// 设置特定主题
this.$store.dispatch('setTheme', 'dark')  // 或 'light'

// 获取当前主题
const currentTheme = this.$store.getters.theme
```

### 3. 主题状态监听
```javascript
computed: {
  ...mapGetters(['theme'])
},
watch: {
  theme(newTheme) {
    console.log('主题已切换至:', newTheme)
  }
}
```

## 主题色彩方案

### 浅色主题 (Light)
- 背景色：白色系 (#ffffff, #f5f7fa)
- 文字色：深色系 (#303133, #606266)
- 边框色：浅灰色系 (#dcdfe6, #e4e7ed)
- 侧边栏：深蓝色系 (#304156, #1f2d3d)

### 深色主题 (Dark)
- 背景色：深色系 (#1a1a1a, #2d2d2d)
- 文字色：浅色系 (#e5e5e5, #cfcfcf)
- 边框色：深灰色系 (#4c4c4c, #414141)
- 侧边栏：深黑色系 (#1f1f1f, #0f0f0f)

## 技术实现

### CSS变量系统
使用CSS自定义属性实现主题切换：
```css
:root {
  --primary-color: #409EFF;
  --bg-color: #ffffff;
  --text-color-primary: #303133;
}

[data-theme='dark'] {
  --bg-color: #1a1a1a;
  --text-color-primary: #e5e5e5;
}
```

### 状态管理
```javascript
// Vuex store
state: {
  theme: Cookies.get('theme') || 'light'
},
mutations: {
  TOGGLE_THEME: (state) => {
    state.theme = state.theme === 'light' ? 'dark' : 'light'
    Cookies.set('theme', state.theme)
  }
}
```

### 动态主题应用
```vue
<template>
  <div id="app" :data-theme="theme">
    <router-view/>
  </div>
</template>
```
