🔧 技术实现细节

1. 核心组件 (HomeCarousel)

- 支持自动播放、手动控制、循环播放 - 触摸手势识别和处理 - 键盘导航支持 - 加载状态和错误处理 - 完整的事件系统

2. 状态管理 (Vuex)

- 数据获取和缓存机制 - 数据过滤和验证 - 错误状态管理 - 加载状态管理

3. API 集成

- 复用现有 homeAdvertise API - 类型过滤 (type: 0) - 状态过滤 (status: 1) - 时间范围验证 - 排序处理

4. 响应式设计

- 桌面端: 1200x400px - 平板端: 992x330px - 小平板: 768x256px - 移动端: 375x200px
📱 移动端优化
  • touch-action: pan-y 优化触摸体验
  • 小屏幕下隐藏控制箭头，主要依靠手势
  • 触摸目标尺寸优化（48px+）
  • 减少动画复杂度以提升性能
  • 支持 prefers-reduced-motion 媒体查询
🎯 使用方法
1. 启动项目：npm run dev 2. 访问首页，轮播图将自动显示在页面顶部 3. 轮播图会自动从 homeAdvertise API 获取数据 4. 支持以下交互方式： - 自动播放（4秒间隔） - 鼠标悬停时显示左右箭头 - 点击指示器跳转到指定图片 - 键盘方向键控制 - 移动端滑动手势
⚙️ 配置选项
<home-carousel :images="carouselImages" // 图片数据 :auto-play="true" // 自动播放 :interval="4000" // 播放间隔 :height="carouselHeight" // 轮播图高度 :show-indicators="true" // 显示指示器 :show-arrows="'hover'" // 箭头显示模式 :loop="true" // 循环播放 :pause-on-hover="true" // 悬停暂停 @change="handleCarouselChange" // 切换事件 @item-click="handleCarouselItemClick" // 点击事件 />
