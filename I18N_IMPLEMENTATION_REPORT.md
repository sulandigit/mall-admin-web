## 使用示例

### 基本翻译
```vue
<!-- 使用翻译 -->
<span>{{ $t('common.button.save') }}</span>

<!-- 菜单翻译 -->
<span>{{ $t('menu.product.title') }}</span>
```

### 语言切换
```vue
<language-switcher 
  :show-icon="true" 
  :show-text="true" 
  @language-changed="onLanguageChanged"
/>
```

### 格式化
```javascript
// 日期格式化
this.$formatDate(new Date()) // 中文: 2024年01月01日, 英文: 01/01/2024

// 货币格式化  
this.$formatCurrency(1234.56) // 中文: ¥1,234.56, 英文: $1,234.56
```
