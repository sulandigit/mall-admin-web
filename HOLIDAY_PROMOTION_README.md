  - 商品数据状态管理

### 3. 核心组件 ✅

#### 倒计时组件 (CountdownTimer.vue)
- 实时倒计时显示
- 支持天/时/分/秒格式
- 活动结束自动触发事件
- 响应式设计适配

#### 优惠券卡片组件 (CouponCard.vue)
- 多种优惠券类型支持（满减券、折扣券、免邮券）

- 领取状态管理
- 热门标识显示

#### 优惠券中心组件 (CouponCenter.vue)
- 优惠券列表展示
- 筛选和排序功能
- 领取确认弹窗
- 分页支持

#### 促销横幅组件 (PromotionBanner.vue)
- 自定义横幅图片和文案
- 集成倒计时显示
- 动画装饰效果
- 点击跳转支持

#### 促销商品展示组件 (PromotionProducts.vue)
- 商品网格展示
- 分类和价格筛选
- 多种排序方式
- 加入购物车和收藏功能

#### 活动配置面板组件 (ActivityConfig.vue)
- 侧边栏配置面板
- 基本信息配置
- 横幅配置
- 优惠券和商品选择
- 实时预览功能
