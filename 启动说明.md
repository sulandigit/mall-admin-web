## 📊 大屏展示功能特色
- **实时数据展示**：销售数据、订单状态、商品排行
- **多维度图表**：柱状图、饼图、折线图、地图
- **响应式设计**：适配不同屏幕尺寸
- **深色科技主题**：专业大屏视觉效果
- **自动刷新**：数据每30秒自动更新

## 🔧 技术栈
- Vue 2.x + Element UI
- ECharts 4.x 图表库
- SCSS 样式预处理
- webpack-dev-server

## 📝 使用说明
1. 访问主应用后，可以通过导航菜单进入"数据大屏"
2. 或直接访问 `/#/dashboard` 进入大屏展示页面
3. 大屏支持全屏显示，适合投影和大屏幕展示
