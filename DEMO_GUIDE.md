# 🎬 商品性能分析报告功能 - 演示指南

## 📺 演示概述

该演示展示了一个完整的商品性能分析系统，基于Vue 2.7.2 + Element UI 2.3.7 + ECharts 4.2.0技术栈构建，具备生产级别的功能特性和用户体验。

### 🎯 演示亮点

- **📊 完整的数据可视化**：多种图表类型展示销售趋势、商品排行、分类分析等
- **🔍 智能筛选系统**：时间范围、分类、品牌、价格等多维度筛选
- **📱 响应式设计**：完美适配桌面端、平板端、移动端
- **⚡ 实时数据更新**：模拟真实业务场景的数据刷新
- **📄 报告导出功能**：支持PDF和Excel格式的分析报告

## 🚀 快速体验

## 🎮 功能演示清单

### 1. 📊 数据可视化演示

#### 销售趋势图表
- **图表类型切换**：折线图 → 柱状图 → 面积图
- **双Y轴显示**：销售额（左轴）+ 销售数量（右轴）
- **交互功能**：缩放、工具提示、图例控制
- **时间粒度**：按天/周/月聚合数据

```javascript
// 演示操作步骤
1. 点击图表上方的"折线图"、"柱状图"、"面积图"按钮
2. 鼠标悬浮查看数据详情
3. 拖拽底部滑块进行数据缩放
4. 点击图例隐藏/显示对应数据系列
```

#### 商品排行榜
- **排序维度**：销售额、销量、利润额
- **TOP数量**：5/10/20可选
- **水平条形图**：直观对比商品表现
- **颜色编码**：彩虹色彩区分不同商品

```javascript
// 演示操作步骤
1. 切换"按销售额"、"按销量"、"按利润"排序
2. 选择显示TOP 5/10/20商品
3. 观察图表动画效果和数据变化
```

#### 分类分析图表
- **饼图/环形图**：展示分类销售占比
- **动态标签**：显示分类名称和百分比
- **交互高亮**：鼠标悬浮突出显示
- **图例联动**：点击图例控制显示

#### 库存状态监控
- **状态分类**：正常库存、库存预警、库存不足、滞销商品
- **颜色编码**：绿色（正常）、黄色（预警）、红色（不足）、灰色（滞销）
- **数据标签**：显示具体数量和占比

### 2. 🔍 筛选控制演示

#### 时间范围选择
- **快捷选择**：今日、昨日、最近7天、最近30天、本月、最近3个月
- **自定义范围**：日期范围选择器
- **对比模式**：同期对比、环期对比

```javascript

3. 启用对比模式查看同比/环比数据
4. 观察图表数据的实时更新
```

#### 商品分类筛选
- **树形结构**：支持多级分类展示
- **多选模式**：批量选择多个分类
- **快速操作**：全选、清空、选择热门分类
- **商品数量**：显示每个分类的商品数量

#### 品牌筛选
- **搜索功能**：实时搜索品牌名称
- **热门推荐**：快速选择热门品牌
- **多选支持**：同时选择多个品牌
- **已选预览**：显示已选择的品牌列表

#### 价格范围滑块
- **可视化选择**：拖拽滑块设置价格范围
- **预设范围**：0-50、50-100、100-300等快捷选择
- **自定义输入**：精确设置最低价和最高价
