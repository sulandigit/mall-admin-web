📊 预测算法说明

线性回归 (Linear Regression)

通过拟合历史数据的线性趋势来预测未来收入。适用于具有明显线性增长或下降趋势的数据。

优势：计算简单，趋势明确

适用场景：稳定增长的业务

移动平均 (Moving Average)

基于最近N个时间段的平均值来预测未来。能够平滑短期波动，反映整体趋势。

优势：平滑噪音，稳定可靠

适用场景：波动较大的数据

指数平滑 (Exponential Smoothing)

对历史数据赋予不同权重，越近期的数据权重越大。能够快速响应趋势变化。

优势：响应及时，适应性强

适用场景：需要快速反应的预测

🚀 使用说明

  1. 启动项目：运行 npm run dev 启动开发服务器
  2. 访问功能：在侧边栏找到"数据分析" → "收入预测"
  3. 设置参数：选择时间范围、数据周期和预测算法
  4. 执行预测：点击"开始预测"按钮查看结果
  5. 分析结果：查看图表趋势、预测精度和增长率
  6. 导出数据：可将分析结果导出为表格文件
