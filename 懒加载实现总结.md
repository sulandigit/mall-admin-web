## 🔍 使用方法

### 基础用法
```javascript
import { lazyLoad } from '@/utils/routeHelper'

{
  path: 'example',
  component: lazyLoad('example/index')
}
```

### 高级用法
```javascript
// 带配置的懒加载
component: lazyLoad('example/index', {
  loading: LoadingComponent,
  error: ErrorComponent,
  delay: 200,
  timeout: 10000
})
```

### 预加载配置
```javascript
import { preloadRoutes } from '@/utils/routeHelper'

// 预加载常用页面
preloadRoutes([
  'home/index',
  'product/list'
], { 
  priority: 'high',
  delay: 1000 
})
```
